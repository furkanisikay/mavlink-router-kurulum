# Katkı Rehberi

Bu projeye katkı vermek istediğiniz için teşekkürler.

## Nasıl Katkı Verebilirim?

1. Repoyu fork edin.
2. Yeni bir dal oluşturun:
   ```bash
   git checkout -b feat/kisa-aciklama
   ```
3. Değişikliklerinizi küçük ve odaklı tutun.
4. Script sözdizimini doğrulayın:
   ```bash
   bash -n kurulum.sh
   ```
5. Değişiklikleri açıklayıcı bir commit mesajıyla gönderin.
6. Pull Request açın ve yaptığınız değişikliği net biçimde açıklayın.

## Kod Standartları

- Değişken/fonksiyon isimlerinde İngilizce teknik isimlendirme kullanın.
- Yorum ve dokümantasyon dilini Türkçe tutun.
- Güvenlik için gizli bilgi (token, parola, anahtar) commit etmeyin.

## Hata Bildirimi

Sorun bildirirken:

- Beklenen davranışı,
- Mevcut davranışı,
- Tekrarlama adımlarını,
- Ortam bilgisini (OS sürümü vb.) paylaşın.
